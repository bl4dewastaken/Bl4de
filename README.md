# BL4DE Portfolio

Static portfolio site for BL4DE.

## Upload

Upload these files to a GitHub repository:

- `index.html`
- `styles.css`
- `script.js`
- `assets/bl4de-showcase.svg`

Then enable GitHub Pages from the repository settings. No build step, package install, or development server is required.

The complete site can also be shared as `bl4de-portfolio.zip`.
